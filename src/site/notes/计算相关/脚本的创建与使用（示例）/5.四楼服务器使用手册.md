日常使用中，只需记住这三个核心命令：`vsubmit`（提交）、`bjobs`（查看）、`bkill`（取消）。

### 1. 提交任务 (`vsubmit`)

在你的个人目录下新建文件夹，准备好 `INCAR`, `POSCAR`, `POTCAR`, `KPOINTS`。

> [!attention] 
>  提交前请务必仔细核对POTCAR和INCAR。特别是对于包含多种元素的体系，**必须严格遵循你设定的元素顺序**。例如，如果 `POSCAR` 中包含 C N Fe S Li 这 5 种元素，请确保 `INCAR` 中的 `LDAUU` 和 `MAGMOM` 参数值数量严格匹配这 5 个元素的顺序。若顺序对不上，排到你的任务时会直接报错秒退，浪费排队时间。

进入计算文件夹，输入提交命令：

Bash

```
vsubmit
```

成功后，你可以直接关闭终端干别的事。

### 2. 查看队列状态 (`bjobs`)

如果想看看前面还有几个人在排队，预估一下自己的任务什么时候开始，随时在终端输入：

Bash

```
bjobs
```

系统会按时间顺序列出当前等待计算的任务清单，同时显示已经计算的任务运行时长。

### 3. 取消任务 (`bkill`)

如果发现结构搭错了或者参数写错了，输入：

Bash

```
bkill
```

- **如果任务还在排队**：系统会直接将其移出队列。
    
- **如果任务正在计算**：系统会二次确认 `是否立即强行终止它？(y/n)`，输入 `y` 即可瞬间结束计算并释放服务器的 48 核资源，守护进程会自动抓取下一个排队任务开始。
    

### 4. 查看当前计算进度

任务开始后，会在相应文件夹下自动生成日志。进入文件夹输入以下命令可实时追踪计算步数（也可直接打开log文件查看）：

Bash

```
tail -f vasp_run.log
```

_(按 Ctrl+C 退出追踪)_
