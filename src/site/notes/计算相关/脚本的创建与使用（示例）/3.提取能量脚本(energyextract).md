### 一、 自动化提取脚本：`energyextract

> [!note] 
> <font color="#f79646">这个脚本会自动遍历当前目录下所有的第一级子文件夹，寻找 `OUTCAR`，提取最后一行 `sigma -> 0` 的能量，并按照文件夹名称排序输出。</font> 

#### 第一步：创建脚本

在 `~/my_scripts/` 目录下创建该文件：

Bash

```
nano ~/my_scripts/energyextract
```


> [!attention] 
> 若没有该文件夹则创建一个 

Bash
```
mkdir -p ~/my_scripts
```

#### 第二步：粘贴以下代码

Python

```
#!/usr/bin/env python3
# -*- coding: utf-8 -*-

import os
import re

def extract_last_energy(outcar_path):
    """从 OUTCAR 中提取最后一个离子步的 sigma -> 0 能量"""
    energy = None
    # 核心修复：兼容了无空格的"entropy="以及带括号的"energy(sigma->0)"
    pattern = re.compile(r"energy\s+without\s+entropy\s*=\s*.*sigma\s*->\s*0\)?\s*=\s*([-+]?\d+\.\d+)")
    
    try:
        with open(outcar_path, 'r') as f:
            for line in f:
                match = pattern.search(line)
                if match:
                    energy = match.group(1)
        return energy
    except Exception:
        return None

def main():
    results = []
    
    # 智能识别：获取当前目录下所有 包含OUTCAR文件 的第一级子文件夹
    subdirs = [d for d in os.listdir('.') if os.path.isdir(d) and os.path.exists(os.path.join(d, 'OUTCAR'))]
    
    if not subdirs:
        print("当前目录下未找到任何包含 OUTCAR 的子文件夹！")
        return

    # 按照字母/数字顺序排序
    subdirs.sort()

    print(f"{'Structure (Folder)':<20} | {'Total Energy (eV)':<20}")
    print("-" * 45)

    for folder in subdirs:
        outcar_path = os.path.join(folder, 'OUTCAR')
        energy = extract_last_energy(outcar_path)
        
        if energy:
            results.append(f"{folder}: {energy}")
            print(f"{folder:<20} | {energy:<20}")
        else:
            print(f"{folder:<20} | 未找到收敛能量，请检查 OUTCAR 格式")

    with open('extracted_energies.txt', 'w') as f:
        f.write("\n".join(results))
    
    print(f"\n提取完成！结果已保存至当前目录下的: extracted_energies.txt")

if __name__ == '__main__':
    main()
```


#### 第三步：给文件添加可执行权限（关键步骤，缺了必报错）

Linux 系统默认不给文本文件执行权限，必须手动添加：

bash

```
chmod +x ~/my_scripts/energyextract
```


#### 第四步：**把脚本库加入系统全局环境变量 PATH（一劳永逸）**

让系统在任何目录都能找到你的`energyextract`命令，执行以下命令写入配置：

bash(若该目录之前已经加入[PATH](1.脚本创建及命名.md#^39e1b2)过则跳过该步骤)

```
echo 'export PATH="$HOME/my_scripts:$PATH"' >> ~/.bashrc
```

#### 第五步：刷新配置生效

bash

```
source ~/.bashrc
```

---


> [!caution] 
>  
脚本文件格式应为`UTF-8`，若在 Windows 下编辑过，可能变为 GBK 编码导致乱码报错，须输入以下命令转换格式：

Bash

```
iconv -f GBK -t UTF-8 ~/my_scripts/energyextract -o ~/my_scripts/energyextract
```

_(这行代码的意思是：把该文件从 GBK 编码读取，转换为 UTF-8，然后重新输出并覆盖原文件)_
